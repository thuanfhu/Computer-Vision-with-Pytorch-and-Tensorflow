## üìñ Course Content

### 1) Python Prerequisites: From Syntax to OOP & Data Science (11h 52m)
* **Basics**: Syntax, Variables, Datatypes, Operators, Control Flow (If/Elif/Else), Loops.
* **Data Structures**: **Lists** (Comprehensions), **Dictionaries**, Sets, Tuples.
* **Functions & Functional Programming**: Lambda, Map, Filter, Generators, Decorators, **Closures**.
* **OOP Mastery**: Classes/Objects, **Inheritance**, **Polymorphism**, Encapsulation, Abstraction, Magic Methods.
* **Data Science Stack**: File I/O, Exception Handling, **Numpy** (Arrays/Math), **Pandas** (DataFrames/Series/Manipulation), Logging.

### 2) Deep Learning Foundations: ANN, Optimizers & Math (6h 23m)
* **Theory**: Perceptron intuition, **ANN** working principles, **Back Propagation**, Chain Rule.
* **Activation Functions**: **Sigmoid**, Tanh, **ReLU** (Leaky/Parametric), ELU, **Softmax**.
* **Loss & Cost**: Regression vs Classification loss functions.
* **Optimizers**: Gradient Descent, **SGD** (Momentum), Adagrad, RMSProp, **Adam**.
* **Problems & Solutions**: Vanishing/Exploding Gradients, Weight Initialization, **Dropout Layers**.
* **CNN Theory**: Convolution operation, **Padding**, Strides, Pooling (Max/Min/Avg), Flattening.

### 3) Computer Vision with OpenCV (OpenCV with Python) (9h 41m)
* **Core Ops**: Reading/Writing Images & Videos, Coordinate Systems, Drawing Shapes/Text.
* **Image Processing**: Color Spaces (RGB/HSV/Gray), **Thresholding**, Resizing, Affine Transformations (Flip/Rotate/Crop).
* **Filters & Features**: Blurring (Gaussian/Median), **Edge Detection** (Sobel, Canny), Histograms (**CLAHE**), Contours.
* **Classical CV Project**: **Haar Cascade** for Face Detection.

### 4) Mastering PyTorch: Tensors to Custom Neural Networks (10h 34m)
* **Tensors**: Creation (Random/Zeros/Ones), Indexing, Manipulation, View/Reshape, Stack, Matrix Aggregation.
* **Models**: Building **Linear Regression**, Multi-Class Classification with Custom NN components.
* **Data Pipeline**: Custom **Dataset** Loader, Data Loaders, Transforms.
* **Training**: Full **CNN Training Loop** implementation using PyTorch on custom datasets.

### 5) Deployment & Visualization: Gradio & CNN Internals (2h 03m)
* **Deployment**: Creating interactive demos with **Gradio**, Hosting on **Hugging Face Spaces**.
* **Deep Visualization**: Understanding CNN Explainer, Tensorspace, Visualizing **Filters** and Feature Maps.
* **Math Deep Dive**: Parameter Calculations, **Receptive Fields**.

### 6) Advanced Image Classification Architectures (3h 57m)
* **Early Models**: **LeNet** (Architecture & Implementation in Keras/PyTorch), **AlexNet**.
* **Modern Giants**: **VGG**, **Inception**, **ResNet** (Residual connections).
* **Technique**: **Transfer Learning** vs Pretrained Models (Feature Extraction vs Fine-tuning).

### 7) Data Augmentation Strategies (0h 30m)
* Concept of Augmentation to reduce overfitting.
* Libraries: **Albumentations** and **Imgaug**.

### 8) Mastering Object Detection: YOLO, R-CNN & Detectron2 (3h 47m)
* **Foundations**: Bounding Boxes, IOU, Object Detection Metrics (mAP).
* **Architectures**: **R-CNN** family (RCNN, Fast RCNN, **Faster RCNN**).
* **State-of-the-Art**: **YOLO** (You Only Look Once) concepts, **Detectron2** framework.
* **Implementation**: Custom Object Detection using **YOLOv11** and Detectron2.

### 9) Mastering Image Segmentation: UNet & Mask-RCNN (4h 55m)
* **Concepts**: Semantic vs Instance Segmentation, Downsampling vs **Upsampling** (Transposed Convolution).
* **Architectures**: **FCNs** (Fully Convolutional Networks), **UNet** (Encoder-Decoder), **Mask-RCNN**.
* **Practice**: Implementing Custom UNet training; Training/Testing **YOLOv11 Instance Segmentation**.

### 10) Capstone Project: YOLO Powered Image Search Application (5h 28m)
* **Overview**: End-to-end full-stack AI application development.
* **Tools**: **Streamlit** for UI/UX, Python Backend.
* **Phases**:
    * **Setup**: Installation & Project Flow design.
    * **Interface**: Building the Search UI with Streamlit.
    * **Logic**: Integrating YOLO for object detection/indexing.
    * **Result**: Displaying and filtering search results based on visual content.
